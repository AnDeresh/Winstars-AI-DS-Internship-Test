# Winstars AI DS Internship Test

This repository contains solutions for two Data Science tasks.

## Structure
```
├── mnist-classification/                           # Task 1
└── ner_image_classification/  # Task 2
```

## Task 1 — MNIST Classification

Three MNIST digit classifiers built with a unified OOP interface:
- Random Forest (`rf`)
- Feed-Forward Neural Network (`nn`)
- Convolutional Neural Network (`cnn`)

See [mnist-classification/README.md](mnist-classification/README.md) for setup and usage.

## Task 2 — Named entity recognition + image classification

An ML pipeline that combines NER and image classification to verify whether a text description matches an animal in an image.

See [ner_image_classification/README.md](ner_image_classification/README.md) for setup and usage.

## Requirements

- Python 3.x
- See `requirements.txt` in each task folder for dependencies.
